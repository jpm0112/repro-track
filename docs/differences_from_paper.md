# Differences from the original paper

artifacts that diverge from the paper. Empty entries are fine — the goal is a
complete audit, not a long list. Reviewers expect transparency about every gap.

Conclusion: ...
Action taken: (e.g. flagged in manuscript §X, retried with override Y, accepted
              as out-of-scope)
